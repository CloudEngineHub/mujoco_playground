We introduce MuJoCo Playground, a fully open-source framework for robot learning built on top of MJX, MuJoCo's open-source GPU-accelerated physics engine. The framework streamlines simulation and sim2real transfer on real robots. With a simple installation process (pip install playground), researchers can train policies within minutes on a single GPU. Playground supports diverse robotic platforms, including quadrupeds, bipeds, humanoids, dexterous hands, and robotic arms, and enables zero-shot sim-to-real transfer from both state and vision inputs. This capability is achieved through an integrated stack comprising an open-source physics engine, batch renderer, and training environments.
